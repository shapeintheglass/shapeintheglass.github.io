CotL Save File Editor

How to use this tool

Use with caution! Save editing can and will potentially break your game.
  1. In Windows Explorer, go to:
    %USERPROFILE%\AppData\LocalLow\Massive Monster\Cult Of The Lamb\saves
  2. Make a backup of the "saves" folder somewhere.
  3. Drag & drop slot_0.json into this window to import your save (for the other slots, use slot_1 or slot_2).
  4. Make your changes and click "Download" or press "Ctrl+S" to download the edited save file.
  5. Replace the file downloaded from this site back into your save folder as either slot_0.json, slot_1.json, or slot_2.json depending on which save slot you wish to use it with.
  6. Run the game!
