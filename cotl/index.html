CotL Save File Editor

How to use this tool

tl;dr Drag save file here, make changes, click "download"
  1. Open Windows Explorer and go to %USERPROFILE%\AppData\LocalLow\Massive Monster\Cult Of The Lamb\saves
  2. Make a backup of this folder somewhere in case you need to go back to it later!!!
  3. Drag & drop either slot_0.json, slot_1.json, or slot_2.json into this window depending on which save slot you wish to edit.
  4. Make your changes and click "Download" or press "Ctrl+S" to download the edited save file.
  5. Place the file downloaded from this site into your save folder as either slot_0.json, slot_1.json, or slot_2.json depending on which save slot you wish to use it with.
  6. Run the game
